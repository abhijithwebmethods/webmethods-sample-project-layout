Sample Setup
SUMMARY
Use this sample to view the EDIINT message and MDN exchange process between two sample partners. To define the two partners, this sample uses two trading partner profiles. Because you can set values to use in the partner profiles, you can have the sample demonstrate an EDIINT message exchange that uses HTTP, HTTPS, SMTP (E-mail), or FTPS.

For use with this sample, the webMethods EDI Module provides sample certificates and private keys that the sample uses when exchanging EDIINT messages between the sample partners.

To use the sample, you set up the profiles; then invoke the wm.EDIINT.sample:send service from the webMethods Developer to execute your EDIINT/MDN message exchange scenario. When you invoke the wm.EDIINT.sample:send service, for the dataFileName input variable, you specify a file to use as the payload of the message. The EDI Module provides two sample EDI payloads that you can select from the drop-down list for the dataFileName input variable. Alternatively, you can specify your own payload.

For more information about exchanging documents using EDIINT, see information about EDIINT in the webMethods EDI Module Concepts Guide and webMethods EDIINT Module User's Guide.
BEFORE YOU BEGIN

  1. If you want to exchange messages that contain payloads that are EDI documents and you want to use the EDI Module to process the EDI documents, be sure you have installed the WmEDI and WmEDIforTN packages of the EDI Module.

    If you want to exchange messages that contain payloads that are EDI documents and do not want to use the EDI Module to process the EDI documents, when configuring EDIINT for the EDI Module, be sure to uncheck the Submit Payload to TN option.

  2. Set up your Trading Networks My Enterprise profile.

  3. If you will set up the sample sender or receiver to exchange EDIINT or MDN messages via SMTP (E-mail), you must enable inbound and outbound e-mail exchange. Similar for FTPS, you must enable FTPS Port. For guidelines, see the webMethods Integration Server Administrator's Guide.
